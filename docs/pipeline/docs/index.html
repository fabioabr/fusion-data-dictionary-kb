Data Dictionary — Oracle Fusion Cloud ERP

2.391 documentos HTML
3 idiomas
Este pipeline automatiza a geração do Data Dictionary do Oracle Fusion Cloud ERP — desde a extração dos metadados dos Excels BICC até a publicação de páginas HTML navegáveis em 3 idiomas. O processo é dividido em 10 etapas sequenciais, onde cada etapa produz artefatos que alimentam a próxima. Algumas etapas requerem LLM (Claude Opus) para geração de conteúdo, enquanto outras são puramente automatizadas via scripts Python.

Etapas do Pipeline
01 Extrair Tabelas Script
Extrai metadados de tabelas dos Excels BICC e gera arquivos .md skeleton com front matter e estrutura de colunas.
02 Extrair PVOs Script
Extrai View Objects (PVOs) dos Excels BICC, classifica por módulo e gera arquivos .json com atributos e tabelas base.
03 Descrever Tabelas LLM Opus
LLM gera descrições ricas: overview, descrição de colunas, categorias semânticas, relacionamentos (FK) e exemplos SQL.
04 Descrever PVOs LLM Opus
LLM gera descrições de negócio em pt-BR para cada PVO, explicando o propósito e os dados extraídos.
05 Criar Arquivos i18n Script
Extrai conteúdo traduzível dos .md e .json, gerando arquivos .i18n.json com campos para PT-BR, EN e ES.
06 Traduzir EN/ES Ollama / Opus
Traduz os arquivos i18n para inglês e espanhol via Ollama local (translategemma:4b) ou Claude Opus em paralelo.
07 Build HTML Tabelas Script
Gera páginas HTML standalone por tabela, com 4 abas (Visão Macro, Colunas, Relacionamentos, SQL), dark/light theme e i18n.
08 Build HTML PVOs Script
Gera páginas HTML standalone por PVO, com 3 abas (Overview, Atributos, Por Tabela), cross-links e i18n.
09 Build Index por Módulo Script
Gera index.html por módulo com tabs (Tabelas + PVOs), busca/filtro, stats e links para cada documento.
10 Build Portal Script
Atualiza o portal principal (docs/index.html) com contadores globais, cards de módulos e estatísticas consolidadas.
