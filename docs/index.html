Este portal centraliza a documentação técnica do projeto Fusion Data Dictionary Discovery — a iniciativa do Banco Patria para construção de uma base de conhecimento corporativa sobre o Oracle Fusion Cloud ERP. O objetivo é mapear, documentar e tornar navegável a estrutura de dados do ERP (tabelas, View Objects, relacionamentos), servindo como alicerce para o discovery de dicionário de dados, classificação de ativos e futura migração para BigQuery. Toda a documentação é gerada automaticamente via um pipeline de 10 etapas e está disponível em 3 idiomas.
Data Dictionary — Oracle Fusion
Dicionário de dados completo do Oracle Fusion Cloud ERP. Documenta tabelas e View Objects (PVOs) de 5 módulos — GL, AP, AR, PO e HCM — com descrições de negócio, estrutura de colunas, relacionamentos (FKs), exemplos SQL e traduções em 3 idiomas. Cada documento é uma página HTML standalone navegável.
