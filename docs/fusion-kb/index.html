Objetivo — Este dicionário documenta as tabelas e View Objects (PVOs) do Oracle Fusion Cloud ERP disponíveis para extração via BICC (BI Cloud Connector). O objetivo é criar uma base de conhecimento navegável que permita consultar rapidamente a estrutura de dados de cada módulo — colunas, tipos, relacionamentos (FKs), queries de uso típico e observações de negócio. Cada tabela possui documentação em 3 idiomas (PT-BR, EN, ES) e links clicáveis entre tabelas relacionadas.

Módulos

General Ledger GL 39 tabelas 59 PVOs
Contabilidade geral — razão contábil, lançamentos, saldos, calendários, combinações de contas (CCID), taxas de câmbio e estrutura de ledgers. Base de toda a estrutura financeira do ERP.
39 core tables GL_BALANCES GL_JE_HEADERS GL_CODE_COMBINATIONS GL_LEDGERS
Accounts Payable AP 37 tabelas 60 PVOs
Contas a pagar — faturas de fornecedores, distribuições contábeis, pagamentos, cheques, retenções (holds), histórico de aprovações e parâmetros do sistema. Integração direta com GL para contabilização.
36 core tables AP_INVOICES_ALL AP_INVOICE_DISTRIBUTIONS_ALL AP_CHECKS_ALL AP_HOLDS_ALL
Accounts Receivable AR 66 tabelas 97 PVOs
Contas a receber — faturas a clientes (invoices, debit/credit memos), recebimentos (cash receipts), ajustes, aplicações de recebimento, schedules de pagamento, aging buckets, cobranças e reconciliação bancária. Inclui tabelas RA_* (transações) e AR_* (operações).
66 core tables Documentado RA_CUSTOMER_TRX_ALL AR_CASH_RECEIPTS_ALL AR_PAYMENT_SCHEDULES_ALL AR_ADJUSTMENTS_ALL
Procurement PO 148 tabelas 289 PVOs
Compras e procurement — ordens de compra (PO headers/lines/distributions), requisições, recebimentos, fornecedores, contratos, negociações (sourcing), catálogos e aprovações. Abrange os prefixos PO_*, PON_*, POQ_*, POR_* e POZ_*.
110 core tables PO_HEADERS_ALL PO_LINES_ALL PO_DISTRIBUTIONS_ALL PO_REQUISITION_HEADERS_ALL
Human Capital Management HCM 755 tabelas 841 PVOs
Gestão de pessoas — colaboradores, cargos, departamentos, organizações, folha de pagamento, benefícios, ausências, talent management, recrutamento e learning. O maior módulo em volume de tabelas, abrangendo os prefixos PER_*, PAY_*, ANC_*, BEN_*, HRT_*, IRC_* e ORA_HRT_*.
804 core tables PER_ALL_PEOPLE_F PER_ALL_ASSIGNMENTS_M HR_ALL_ORGANIZATION_UNITS_F PAY_PAYROLL_ACTIONS
