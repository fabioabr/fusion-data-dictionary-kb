# PaymentSchedulePVO

## 📌 Visão Geral

Extrai os cronogramas de pagamento (parcelas) das transações de Contas a Receber, incluindo valor original, saldo em aberto, data de vencimento e status. Central para análise de aging, gestão de inadimplência, projeção de fluxo de caixa e cobrança.
