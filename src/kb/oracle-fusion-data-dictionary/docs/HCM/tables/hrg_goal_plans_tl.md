# HRG_GOAL_PLANS_TL

## 📌 Visão Geral

Tabela de **traduções** de planos de metas (base). Padrão Oracle MLS.

---

## 🧠 Propósito de Negócio

Esta tabela é utilizada nos seguintes processos:

- **Internacionalização:** Traduções em múltiplos idiomas.

## 📎 Uso Típico

### Traduções
```sql
SELECT tl.NAME, tl.DESCRIPTION
FROM   HRG_GOAL_PLANS_TL tl
WHERE  tl.GOAL_PLAN_ID = :p_id AND tl.LANGUAGE = USERENV('LANG');
```
