Sistema: i18n (data-i18n + switchLang)
O sistema de traducao usa atributos data-i18n nos elementos HTML e um dicionario JS (T) com chaves para cada idioma (pt, en, es). A funcao switchLang(lang) percorre todos os elementos com data-i18n e atualiza o innerHTML. Placeholders usam data-i18n-placeholder e tooltips usam data-i18n-title.
Exemplo: Elementos traduzidos (use o seletor de idioma no header)
Exemplos de data-i18n
Atributo Chave Resultado (troque o idioma para ver)
data-i18n i18n_example_title Titulo do Documento
data-i18n i18n_example_desc Esta descricao muda conforme o idioma selecionado no header.
data-i18n-placeholder search_placeholder
Como usar: Adicione data-i18n="chave" no elemento HTML. No JS, adicione a chave no dicionario T com traducoes para pt, en e es. Chame switchLang('en') para trocar o idioma.
